## 💻 About the project
Website where you can connect and buy cars online. Project done in a group of 2 people.

## ⚙️ Functionalities
- Log in/Sign up
- Buy cars
- Add cars to cart
- Remove cars from cart

## 🛠 Technologies
- Languages : `PHP`, `Twig`, `CSS`
- Frameworks : `Symfony`
- Architecture : `MVC`
- Databases : `SQL`
